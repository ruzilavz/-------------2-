3. ИИ-СИСТЕМА НА САЙТЕ
Главный чат (ChatKit)

Использует OpenAI ChatKit.
Функции:

отвечает пользователю;

пишет в стиле: дружелюбный, творческий, помогает в музыке, играх и разработке;

может вести диалог двух агентов.

Агенты:

Agent A — аналитик

логика, структура, коды, разработка игр

Agent B — творческий музыкант

тексты, рифмы, образы, ощущения

Формат:

[Agent A]: ...
[Agent B]: ...
[Result]: итог для пользователя

Генераторы

генератор текстов песен (structure-aware, оригинальные строки)

генератор обложек по стилю артиста

генератор SVG-персонажей

игровой ассистент (механика Idle Game)

6. ЧЕГО ОТ ИИ ОЖИДАЕТСЯ (главные правила)

ИИ должен:

понимать структуру сайта;

помогать дополнять разделы;

писать тексты, описание, промпты, идеи;

упрощать разработку (структуры файлов, код, документация);

не путать разные проекты: музыка / игра / чат;

соблюдать стиль AVZALØV — тёмный, глубокий, современный;

не менять структуру игры без команды;

отвечать кратко, если нужен код — давать чистый код;

в чат-агентах использовать формат A/B/Result при запросах.

ИИ не должен:

удалять важную логику игры

переписывать макеты без указаний

путать SVG-героев

менять данные артиста

1. ОБЩЕЕ НАЗНАЧЕНИЕ САЙТА

Сайт AVZALØV — это интерактивная многофункциональная платформа музыканта, битмейкера, автора и разработчика Рузиля Авзалова.
Сайт объединяет музыку, авторский бренд, AI-инструменты, игровую систему, визуальное портфолио и собственную валюту RUZCOIN.

Основные цели сайта:

продвижение музыки и релизов AVZALØV;

игровой функционал (RUZCOIN, уровни, достижения);

AI-чат и генераторы контента;

персональный кабинет пользователя;

магазин и пополнение внутренней валюты;

показ клипов, треков и проектов.

2. СТИЛИСТИКА И UX

Сайт выполнен в стиле:

глубокий "dark-mode";

неоновые акценты (синий/фиолетовый/оранжевый);

мягкие блики и свечения;

стеклянные поверхности (glass UI);

много теней, объемов, 3D-эффектов;

плавные анимации карточек, наклоны, параллакс.

Все UI-элементы визуально сложные:
градиенты, border-glow, hover-лифты, мягкие подсветки, скругления.
